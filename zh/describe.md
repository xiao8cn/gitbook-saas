#目录描述

***

####项目结构

```js
  +─build                             // 编译项目用js和webpack配置
  +─config                            // webpack和npm引用的一些环境变量配置
  +─libs                              // 引用的第三方js和css等
  +─schema                            // cms用schema用的数据
  +─src                               // 项目代码
  +─static                            // 项目静态资源
  ├─.babelrc                          // babel开发环境配置
  ├─.eslintignore                     // eslint 忽略配置
  ├─.eslintrc.js                      // eslint 基础配置
  ├─.eslintrc.json                    // eslint json配置项
  ├─gulpfile.js                       // gulp 内容
  ├─package.json                      // npm 配置
  ├─README.md
```

> 开发人员只需要关注src目录下的内容即可
